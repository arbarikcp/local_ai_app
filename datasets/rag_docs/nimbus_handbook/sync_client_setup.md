# Sync Client Setup

The Nimbus desktop sync client is available for macOS 12+, Windows 10+, and major Linux
distributions via .deb and .rpm packages. After installing, sign in with your Nimbus account
and choose a local folder to sync — by default `~/Nimbus` on macOS/Linux and
`C:\Users\<you>\Nimbus` on Windows.

Selective sync lets you choose which cloud folders sync locally, useful when local disk space
is smaller than your Nimbus storage plan. Sync conflicts (the same file edited offline on two
devices) create a second file named `filename (conflicted copy).ext` rather than silently
overwriting either version.
