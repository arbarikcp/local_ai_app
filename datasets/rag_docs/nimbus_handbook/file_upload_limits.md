# File Upload Limits

The maximum single-file upload size is 5GB on Free and Personal plans, and 50GB on Business
and Enterprise plans. There is no limit on the total number of files, only on total storage
consumed by your plan.

Uploads over 100MB automatically use resumable chunked upload, so an interrupted connection
resumes from where it left off rather than restarting. Files are scanned for malware on
upload; infected files are quarantined and the uploader is notified by email within minutes.
