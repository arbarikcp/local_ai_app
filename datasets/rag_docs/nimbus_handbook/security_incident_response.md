# Security Incident Response

If you suspect unauthorized access to your Nimbus account, immediately reset your password
(see password_reset.md) — this signs out all active sessions on every device. Then review
Account Settings > Security > Active Sessions to confirm no unrecognized devices remain.

Nimbus's security team investigates every reported incident and notifies affected users
within 72 hours of confirming unauthorized access, in line with standard breach-notification
practice. Enterprise customers with a signed Data Processing Agreement receive a detailed
incident report within 5 business days of confirmation.
