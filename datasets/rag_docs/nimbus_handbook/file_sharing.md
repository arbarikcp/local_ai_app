# File Sharing

You can share any file or folder via a link with view-only or edit permissions. Shared links
can optionally require a password and can be set to expire after a specific date. By default,
shared links never expire unless you set an expiration.

Free plan users can create up to 10 active shared links at a time; paid plans have no limit.
Anyone with an edit-permission link can modify the file without needing a Nimbus account,
unless "Require sign-in to edit" is enabled in the share settings.
