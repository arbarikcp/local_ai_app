# Mobile App Guide

The Nimbus mobile app is available for iOS 15+ and Android 10+. The mobile app supports
automatic camera-roll backup, which can be restricted to Wi-Fi only in Settings > Backup to
avoid mobile data charges.

Offline access to specific files can be enabled per-file by tapping the download icon; files
marked for offline access are re-synced automatically when the device regains connectivity.
The mobile app does not currently support folder-level permission management (see
folder_permissions.md) — that requires the desktop or web app.
