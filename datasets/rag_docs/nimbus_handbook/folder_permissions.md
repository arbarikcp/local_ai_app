# Folder Permissions

On Business and Enterprise plans, folders support granular permissions: Owner, Editor,
Commenter, and Viewer, assignable per workspace member. Personal and Free plans only support
whole-account sharing via links (see file_sharing.md), not per-member folder permissions.

Permission changes apply immediately to all files within a folder, including files added
after the permission was set. Removing a member from a workspace immediately revokes their
access to all folders, even ones they were explicitly granted Editor access to.
